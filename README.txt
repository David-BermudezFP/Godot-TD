POKE TD

Juego estilo Tower Defense infinito con estética Pokémon.

-18 enemigos diferentes de 4 tipos diferentes (Normal, Planta, Agua, Fuego) con diferentes estadísticas de vida y velocidad.

-Un único mapa con 4 rutas diferentes para que los enemigos se dirijan hacia la puerta

-API REST que guarda la información de Usuario y puntuación de la partida y mostrará las 5 mejores partidas jugadas al final de una partida en la pantalla de GAME OVER

-3 pistas de sonido diferentes para los tutoriales, main menú y la partida con volumen configurable desde opciones o pulsando ESC en una partida

-Menú de opciones (accesible desde el main menú pulsando opción o con ESC durante una partida) en el que se pueden modificar tanto el volumen como la pantalla

-Menú de tutorial para aprender las mecánicas básicas del juego
